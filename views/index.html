Exercise Tracker

Create a new user

POST /api/exercise/new-user

I can create a user by posting form data username to /api/exercise/new-user and returned will be an object with username and _id.

Add exercises

POST /api/exercise/add

I can add an exercise to any user by posting form data userId (_id), description, duration, and optionally date to /api/exercise/add. If no date is supplied, it will use the current date. The user object with exercise fields added should be returned.

Get all exercises

GET /api/exercise

Get all users

GET /api/exercise/users

I can get an array of all users by getting api/exercise/users with the same info as when creating a user.

Get one user's exercise logs

GET /api/exercise/log?{userId}[&from][&to][&limit]

I can retrieve a full exercise log of any user by getting /api/exercise/log with a parameter of userId (_id). The user object with added exercises log and a count of all exercises will be returned.

I can retrieve part of the exercises log of any user by also passing along optional parameters of from, to, and/or limit. (Date format yyyy-mm-dd, limit = int)

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
